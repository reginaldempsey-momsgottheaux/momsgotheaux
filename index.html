How to use

1. Upload songs in the Songs tab

2. Tap the clip icon to open the full-screen clip editor

3. Drag handles to set exact start and end points

4. Create events in the Events tab

5. Assign songs to each event category

6. Game tab — tap buttons during the game!

Songs cycle without repeating until all are played.
