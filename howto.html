Setup Guide

Get Your Songs In

Your songs are on Spotify. Let's get them into the app. Pick your path — takes about 15 minutes either way.

🏒 Easy Path
Step by step, no experience needed
⚡ Power Path
Already comfortable with Terminal
1
Export Your Spotify Playlist
We need a list of your songs. Exportify grabs it from Spotify in one click — no install needed, just a website.

1. Go to exportify.net
2. Click Log in with Spotify
3. Find your hockey playlist and click Export
4. Save the CSV file to your Downloads folder
What's a CSV? Just a spreadsheet file with your song names and artists. Nothing scary.
2
Download a Free Helper App
We need two small free tools on your computer. Don't worry — this is a one-time setup.
Mac
Windows
Step 2a — Install Homebrew (the Mac app installer)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
1. Press Command + Space, type Terminal, hit Enter
2. Paste the line above and hit Enter
3. Enter your Mac password when asked (you won't see it typing — that's normal)
4. Wait about 5 minutes
Step 2b — Install yt-dlp and ffmpeg
sudo curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp_macos -o /usr/local/bin/yt-dlp && sudo chmod a+rx /usr/local/bin/yt-dlp && brew install ffmpeg
Tip: These only need to be installed once ever. Next time you want to download songs you skip straight to Step 3.
1. Go to python.org/downloads and install Python
2. During install, check "Add Python to PATH" — important!
3. Open Command Prompt (search it in Start menu)
4. Paste this and hit Enter:
pip install yt-dlp
5. Then install ffmpeg from ffmpeg.org — download the Windows build and add it to your PATH, or use:
winget install ffmpeg
3
Turn Your Song List Into MP3s
Now we use your CSV file to download all the songs. This command reads every song name from the file and finds it on YouTube automatically.
Mac
Windows
In Terminal, paste this — replacing the filename with your actual CSV filename:
yt-dlp --extract-audio --audio-format mp3 --audio-quality 0 -o "%(title)s.%(ext)s" --batch-file <(awk -F',' 'NR>1 && $2!="" && $4!="" {gsub(/"/, "", $2); gsub(/"/, "", $4); print $4 " - " $2}' ~/Downloads/YOUR_PLAYLIST.csv | while read line; do echo "ytsearch:$line"; done)
Replace YOUR_PLAYLIST.csv with your actual filename — e.g. New_Game_Day.csv
This will take 20–40 minutes for a full playlist. Leave Terminal open and grab a coffee. Songs save to your home folder (the one with your name in Finder).
In Command Prompt, paste this — replacing the filename with your actual CSV filename:
for /f "tokens=2,4 delims=," %A in ('more +1 "%USERPROFILE%\Downloads\YOUR_PLAYLIST.csv"') do yt-dlp --extract-audio --audio-format mp3 --audio-quality 0 -o "%%(title)s.%%(ext)s" "ytsearch:%B - %A"
Replace YOUR_PLAYLIST.csv with your actual filename.
4
Upload to Mom's Got the Aux
Once your MP3s are downloaded:

1. Open momsgotheaux.vercel.app
2. Create an account or sign in
3. Go to the Songs tab
4. Tap Upload Songs and select all your MP3 files at once
5. The app uploads them one by one — leave it running, takes a few minutes
Your songs are stored securely in the cloud — they'll be there every time you open the app, on any device.
5
Clip Your Songs
This is the fun part. For each song, tap the clip icon to set exactly which part plays — maybe just the chorus drop, or the hype intro.

1. Tap the scissors icon next to any song
2. Drag the green handle to set where it starts
3. Drag the red handle to set where it ends
4. Hit Save Clip
Pro tip: Most DJ clips are 10–30 seconds. Pick the most hype moment of the song — usually the chorus or the drop.
6
Assign Songs to Events
Go to the Events tab and assign your songs to game moments — Goal, Icing, Power Play, etc. Add as many songs per event as you want. The app cycles through them so the same song never plays twice in a row.

You're Ready to Rock the Rink! 🏒

Open the app on your phone or tablet at the game, go to the Game tab, and press buttons when the whistle blows.
